## Get started:

- cd into this folder: `cd ep_01_setup`

### Install Dependencies:

#### Windows

- pip install fastapi uvicorn
  or
- pip install -r requirements.txt

#### Mac / Linux

- pip3 install fastapi uvicorn
  or
- pip3 install -r requirements.txt

### Run File:

- uvicorn app:app --reload
